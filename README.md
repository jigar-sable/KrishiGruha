# KrishiGruha
A platform made for farmers/customers to buy/sell their products online at the best prices.

## Technologies used :
Android / IOS APK - React Native  
Backend   -   Node.js (Hosted on Heroku server)  
Database  -   MongoDB (Cloud Atlas)
